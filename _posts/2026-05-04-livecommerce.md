## 프로젝트 소개

라이브 방송을 통해 상품을 판매하는 라이브커머스 플랫폼입니다.

**Live**: [live.lemuel.co.kr](https://live.lemuel.co.kr)

## 핵심 기능

- 실시간 방송 스트리밍
- 방송 중 상품 등록/관리
- 실시간 채팅
- 상품 CRUD + 이미지 관리

## 기술 스택

Spring Boot 4 / Java 25 / PostgreSQL / Redis / React + Vite
