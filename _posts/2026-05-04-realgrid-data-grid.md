## 프로젝트 소개

RealGrid 엔터프라이즈 데이터 그리드 컴포넌트를 Spring Boot + React로 연동한 프로젝트입니다. 대용량 데이터의 가상 스크롤, 필터링, 정렬, 편집을 지원합니다.

**Live**: [realgrid.lemuel.co.kr](https://realgrid.lemuel.co.kr)

## 핵심 기능

- 가상 스크롤 (수만 건 데이터)
- 인라인 편집 + 일괄 저장
- 동적 테이블 생성/관리
- 고객 데이터 CRUD

## 기술 스택

Spring Boot 4 / Java 25 / H2 / React / RealGrid
